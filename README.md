# A2 Luxe Shopping Store

A2 Luxe Shopping Store is a modern Android e-commerce application developed using Java and Firebase. The application provides a complete shopping experience with secure authentication, cart management, wishlist functionality, multilingual support, and theme customization.

## Features

* Firebase Authentication
* Google Sign-In
* Email Verification
* Product Browsing
* Add to Cart
* Wishlist Management
* User Profile Management
* Settings Page
* Urdu Language Support
* Dark Mode & Light Mode
* Firebase Database Integration
* Responsive User Interface

## Technologies Used

* Java
* Android Studio
* Firebase Authentication
* Firebase Firestore / Realtime Database
* Remote Configration
* Material Design Component

## Installation

1. Clone the repository.
2. Open the project in Android Studio.
3. Connect Firebase.
4. Sync Gradle files.
5. Run the application on an Android device or emulator.
